How to Select and Export Project Preferences

How to Select and Export Your Project Preferences

  1. Browse the project list
    Review the available projects shown on the following page.
  2. Select a project
    Click on any project card to add it to your shortlist.
  3. View your selected projects
    Your chosen projects can be seen by clicking Show Selected Projects . You can remove a project at any time before exporting by clicking it again.
  4. Assign a preference rank
    For each selected project, choose a unique preference ranking from 1–12, where:
    • 1 = highest preference
    • 12 = lowest preference
  5. Export to CSV
    Once all preferences are complete, click Export Project Preference CSV to download your submission file.

Preference Rules

Before exporting, the system checks that:

  • You have selected exactly 12 projects (not more or less)
  • Every selected project has a preference rank assigned
  • Preference ranks are unique (no duplicate rankings)
  • A project has not already been assigned to another student, unless that project has capacity for more than one student

If any of these rules are not met, export will be blocked until the issue is corrected.

Filters

Use the filter controls above the project list to narrow what is shown:

  • Show Selected Projects – displays only projects you have selected
  • Show Unassigned Only – displays only projects that have not been pre-allocated.
  • Search / Category filters – filter projects by keyword, supervisor, category, or Fields.

Filters only affect what is visible on screen — they do not remove projects from your selected list.
